# Latitude AI - Senior Autonomy Integration Engineer Candidate Vision

An independent candidate campaign by Russell Dudek for Latitude AI's Senior Autonomy Integration Engineer role in Pittsburgh.

## Candidate thesis

**Make the system agree with itself.**

The campaign argues that integration is not the phase after engineering. It is the mechanism that converts specialized engineering velocity into one launchable system through reproducible behavior, traceable configuration, regression evidence, and clear ownership across hardware and software boundaries.

## Interaction

The System Coherence Frame opens with an illustrative new-platform bring-up scenario. Visitors can compare platform bring-up, cross-domain regression, intermittent failure, and configuration drift. Each choice changes the affected layers, closure state, reproduction logic, authority boundary, system change, and release evidence. A reset returns to the labeled baseline.
